# Online Retail Sales Analytics Dashboard

## Project Overview

This project presents an interactive Power BI dashboard built using the Online Retail Dataset (2010–2011). The dashboard provides insights into sales performance, customer activity, product performance, and geographic revenue distribution.

The objective of this project is to transform raw retail transaction data into actionable business insights through data cleaning, modeling, DAX calculations, and dashboard visualization.


## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dataset

Source: Online Retail Dataset (2010–2011)

The dataset contains retail transaction records including:
- Invoice Number
- Product Description
- Quantity Sold
- Unit Price
- Customer ID
- Country
- Invoice Date

## Key Insights

- The business generated approximately **$10.67 million** in total revenue.
- Revenue increased significantly during **October–December**, indicating strong seasonal demand.
- A small group of products contributed a large share of total revenue.
- The **United Kingdom** generated the highest revenue among all countries.
- Customers placed multiple orders, suggesting repeat purchasing behavior.
- Revenue peaked during the holiday season, highlighting the importance of inventory and marketing planning in Q4.
